# 🎙️ AI-ML Researcher Speech Audio Project

## 📌 Overview
This repository contains the implementation of an **AI/ML Speech Processing Pipeline** developed as part of a research-oriented task provided by a company. The project focuses on **Automatic Speech Recognition (ASR)**, data cleaning, evaluation, and model improvement techniques.

This project was assigned as part of a **company evaluation task** for an AI/ML Researcher role.

## 🧠 Key Features

### 🔊 Speech-to-Text Processing
- Used Whisper-small model for ASR
- Processed Hindi conversational audio

### 🧹 Data Cleaning Pipeline
- Cleaned noisy ASR outputs
- Normalized:
  - Hindi numbers → numeric format  
  - Mixed Hindi-English words  
  - Special characters & noise  

### 📊 Evaluation Metrics
- Implemented **Word Error Rate (WER)**
- Compared:
  - Raw ASR output  
  - Cleaned output  

### 🧩 Lattice-based Evaluation
- Generated multiple hypotheses for better evaluation
- Improved flexibility over strict ground truth comparison  

### 📁 Dataset Handling
- Processed JSON transcription files  
- Structured dataset for training & evaluation
